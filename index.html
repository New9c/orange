Game is simple, you use your Haute42 to block the wack ass mixup.

It gets weird, but you only need to know three attacks:

Normal:

Block by holding the block button, you should alway default to doing so.

Jumpy:

Dude may jump up and attack. If that happens, block high by NOT holding the block button.

ORANGE:

Despite being on the ground, dude may charge up the ultimate attack: ORANGE. You have to react as you would to a jumpy attack, look out for ORANGE!
